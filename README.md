# AWS Three-Tier Architecture

## Project Overview

This project demonstrates a secure and highly available three-tier web application architecture on AWS.

## AWS Services Used

- Amazon VPC
- EC2
- Application Load Balancer
- Auto Scaling
- Amazon RDS
- Security Groups
- Route 53

## Features

- High Availability
- Auto Scaling
- Private Database
- Public Load Balancer
- Secure Networking
